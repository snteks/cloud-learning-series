# Troubleshooting

If get error like this because deleted in console:

╷
│ Error: Get "http://localhost/apis/rbac.authorization.k8s.io/v1/clusterroles/eks-console-dashboard-full-access-clusterrole": dial tcp [::1]:80: connect: connection refused
│
│ with module.eks_cluster.kubernetes_cluster_role.cluster_role,
│ on modules/eks/cluster/main.tf line 206, in resource "kubernetes_cluster_role" "cluster_role":
│ 206: resource "kubernetes_cluster_role" "cluster_role" {

then you need to locally run:
`terragrunt state list`
`terragrunt state rm module.eks_cluster.kubernetes_cluster_role.cluster_role`
`terragrunt state rm module.eks_cluster.kubernetes_cluster_role_binding.cluster_role_binding`

then you can run the terraform again
